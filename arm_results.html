When performing an association rule analysis, the thresholds (or support, confidence, and improvement thresholds) are related to the quality of the association rules, so setting these thresholds is a significant part of the analysis. Since the goal of association rule analysis is to find the characteristics of a car associated with each price category of a vehicle, a high threshold can be good for getting more general and reliable association rules. However, this may result in fewer association rules to analyze the data. Hence, it needs to be adjusted depending on the analysis's purpose and the data's nature. In this project, support is set to 0.95 and confidence to 0.1. Although support is a relatively low value, it is set to this value to derive enough association rules for analysis. Although confidence is set to 0.1, obtaining association rules with higher confidence in the actual data is possible. Lift is not set as a separate threshold but is used to analyze the association of each attribute through the results obtained. Finally, minlen is set to 3 to find association rules with more diverse features.
